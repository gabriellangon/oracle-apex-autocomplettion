# Chrome Web Store Listing

## Extension Name
**APEX Autocomplete - SQL/PL/SQL IntelliSense**

## Short Description (132 characters max)
SQL, PL/SQL and Oracle APEX API autocomplete for Oracle APEX editors. Smart code completion, formatting, and language switching.

## Detailed Description

### English

**APEX Autocomplete** brings powerful IntelliSense-style code completion to Oracle APEX's Monaco code editors.

**Key Features:**

- **SQL Autocomplete** - Complete SQL keywords, clauses, and syntax as you type
- **PL/SQL Autocomplete** - Smart completion for PL/SQL keywords, built-in functions, and procedural constructs
- **Oracle APEX API Autocomplete** - Full support for 500+ APEX PL/SQL APIs including:
  - APEX_UTIL
  - APEX_PAGE
  - APEX_APPLICATION
  - APEX_COLLECTION
  - APEX_MAIL
  - APEX_JSON
  - APEX_WEB_SERVICE
  - APEX_DEBUG
  - And many more...

- **Smart Code Formatting** - Automatic PL/SQL indentation and formatting with proper alignment
- **Language Switching** - Easily switch between SQL and PL/SQL modes via the extension popup
- **Variable Detection** - Recognizes declared variables in your code and suggests them

**How It Works:**

1. Install the extension
2. Navigate to any Oracle APEX page with a Monaco code editor
3. Start typing - autocomplete suggestions appear automatically
4. Press Tab or Enter to accept a suggestion
5. Click the extension icon to switch languages or access settings

**Perfect for:**
- Oracle APEX developers
- PL/SQL programmers
- Database administrators working with APEX
- Anyone writing SQL/PL/SQL in Oracle APEX

**Privacy:**
This extension runs entirely in your browser. No data is collected or sent to external servers.
